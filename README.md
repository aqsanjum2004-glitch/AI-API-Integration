# AI API Integration Project

## Description
This project demonstrates integration of multiple AI APIs:
- Groq
- Hugging Face
- Cohere
- Ollama

Note:
- OpenAI API is excluded due to payment requirements.
- Google Gemini API is excluded due to quota limitations.

## Features
Each program:
- Accepts user input
- Sends request to API
- Displays response
- Handles errors properly

## Files Included
- Python files for each API
- requirements.txt
- Screenshots of outputs
- Outputs.pdf

## Setup Instructions

1. Install dependencies:
pip install -r requirements.txt

2. Set environment variables:

Example:
export COHERE_API_KEY=your_key

3. Run any file:
python cohere_api.py

## Outputs
All outputs are included in:
- Screenshots folder
- Outputs.pdf
